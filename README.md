# sentinel-ai
An AI-powered Cyber Security Agent for authorized security assessments, vulnerability analysis, network monitoring, AI-assisted security guidance, and automated reporting.
## Features

- AI Security Assistant
- Vulnerability Analysis
- Website Security Scanner (authorized targets only)
- SSL/TLS Checker
- DNS & WHOIS Lookup
- CVE Intelligence
- Security Reports
- JWT Authentication
- RBAC
- React Dashboard
- Docker Deployment

## Tech Stack
- FastAPI
- React
- PostgreSQL
- SQLAlchemy
- Docker
- Redis
- Celery
- OpenAI / Ollama

## Disclaimer

This software is intended only for defensive cybersecurity, education, and authorized security testing. Use only on systems you own or have permission to assess.
